Selected Projects

May 2021

Rapid Visual Search Using CNN Fingerprinting

The availability of satellite/aerial imagery has increased in recent years, and analyzing this type of data is invaluable for many modern tasks: optimizing crop yield, predicting weather, monitoring infrastructure, etc. As the total amount of aerial imagery data exceeds the petabyte scale, it is necessary to have efficient methods to analyze and compress this type of data. This paper presents one such method: using convolutional neural networks to compress large datasets of aerial images into low-dimensional fingerprints, which enables rapid similarity search between a query image and the rest of the dataset. .

Aug 2020

MIT Media Lab: Saving Face Project

This project used a SONAR-inspired approach to detect proximity between hands and face and warns the user with an alarm when they get too close. I developed an ML model to process the signals generated from the Ultrasound reflecting off an approaching hand. The goal of the model is to be able to accurately classify whether a sequence of signals which are returned to the microphone indicates whether a hand is touching the face or not. I improved the robustness and accuracy of the model by introducing new types of training data and noise and streamlining the data generation process to strengthen the model.
